# Dataset Information

This project uses the HAM10000 dermoscopic image dataset
containing ~10,000 images across 7 skin lesion classes.

Due to size constraints, the dataset is not included in this repository.

Google Drive link:
https://drive.google.com/drive/folders/1d3yW1FXD_6x55OF0IMV0-05xua_9SiZJ?usp=sharing

Dataset source:
https://www.kaggle.com/datasets/kmader/skin-cancer-mnist-ham10000

To run the project:
- Download the dataset
- Upload it to Google Drive
- Mount Drive in Google Colab
- Update dataset path if required
